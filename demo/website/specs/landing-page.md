# Feature: a basic marketing landing page for "Loopwear"

## Summary
Build a single static landing page (`index.html`) plus a stylesheet
(`styles.css`) for a fictional clothing brand called **Loopwear**. It must be a
complete, self-contained page a browser can open directly from disk.

## Acceptance criteria
- AC-1: `index.html` exists at the repo root and has a non-empty `<title>`.
- AC-2: it links the stylesheet with `<link rel="stylesheet" href="styles.css">`.
- AC-3: it has a `<nav>` containing links whose text includes **Home**, **About**, and **Contact**.
- AC-4: it has a hero section with a single `<h1>`.
- AC-5: every `<img>` (if any) has a non-empty `alt` attribute.
- AC-6: it has a `<footer>` whose text contains a copyright notice (the "©" symbol or the word "copyright").
- AC-7: `styles.css` exists at the repo root and is non-empty.

## Notes
Keep it simple and semantic. No build step, no frameworks, no external network
requests — a browser opening `index.html` from disk must render the finished page.
